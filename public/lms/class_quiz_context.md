2. Nmap Purpose

What does the nmap command primarily do?

A) Cracks passwords
B) Encrypts hard drives
C) Network exploration and security auditing (Port scanning)
D) Installs a backdoor

Answer: C) Network exploration and security auditing

3. Nmap OS Detection

Which flag is used with nmap to perform an OS detection scan?

A) -sV
B) -O
C) -p-
D) -Pn

Answer: B) -O

4. Change Directory Command

In a Linux terminal, what command is used to change the current directory?

A) chdir
B) move
C) cd
D) ls

Answer: C) cd

5. whoami Command

What is the purpose of the whoami command?

A) To find the owner of a website
B) To display the current logged-in username
C) To list all users on a network
D) To change the user password

Answer: B) To display the current logged-in username

6. Passive Reconnaissance

Which of the following is considered passive reconnaissance?

A) Running a vulnerability scan
B) Searching a company's public LinkedIn profiles
C) Brute-forcing a login page
D) Pinging a server

Answer: B) Searching a company's public LinkedIn profiles

7. sudo Meaning

What does the sudo command stand for?

A) System User Direct Open
B) Secure User Data Only
C) Superuser Do
D) Super User Delete Option

Answer: C) Superuser Do

8. Viewing File Contents

Which command is used to view the contents of a file named passwords.txt?

A) read passwords.txt
B) open passwords.txt
C) cat passwords.txt
D) view passwords.txt

Answer: C) cat passwords.txt

9. Network Sniffing Tool

Which tool is commonly used for sniffing network traffic?

A) John the Ripper
B) Wireshark
C) Metasploit
D) Burp Suite

Answer: B) Wireshark

10. First Phase of Ethical Hacking

What is the first phase of a professional ethical hack?

A) Gaining Access
B) Maintaining Access
C) Reconnaissance (Footprinting)
D) Clearing Tracks

Answer: C) Reconnaissance (Footprinting)
